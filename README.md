# 高考：普通*高*等学校招生全国统一*考*试

*（自1991年起，国家教委出版的高考相关书籍统一使用此全称）*

# 1. 高考改革史
   - 1952：统一高考制度建立
   - 1952-1957：统一高考下分省与统一的争论
   - 1958：分省命题
   - 1959-1965：全国统一试卷
   - 1966-1976：高考停止
   - 1977：高考恢复，*因故*分省命题
   - 1978-1984：全国统一试卷
   - 1985：上海自主命题，广东启动***标准化***改革
   - 1986-1990：标准化改革继续
   - 1991：广东回归全国卷，三南试验启动
   - 1992：三南试验继续
   - 1993：三南试验结束，3+2方案开始实行
   - 1994：3+2方案继续推广
   - 1995：3+2方案推广完成
   - 1996：3+2方案与上海方案并存
   - 1997：天津，山西，江西启动***新课程***改革
   - 1998：新课程改革继续
   - 1999：广东启动3+X改革
   - 2000：广东调整3+X
   - 2001：河南加入3+X
   - 2002：北京自主命题，江苏，广西加入3+X，3+X改革完成
   - 2003：辽宁加入实验
   - 2004：重庆，福建，广东，湖北，湖南，江苏，辽宁，天津，浙江自主命题，***新课标***改革启动
   - 2005：山东，江西自主命题，新课程改革完成
   - 2006：安徽，四川，陕西自主命题
   - 2007-2009：教育部为海南，宁夏单独命题，新课标改革继续
   - 2010：海南，宁夏卷结束，并入新课标卷
   - 2011-2013：新课标改革继续
   - 2014：新课标改革继续，***新高考***改革启动
   - 2015：新课标改革完成，江西，辽宁停止自主命题
   - 2016：重庆，福建，广东，湖北，湖南，安徽，陕西，停止自主命题
   - 2017：四川停止自主命题
   - 2018：山东停止自主命题
   - 2019：新高考改革继续
   - 2020：新高考全国卷出现
   - 2021：江苏停止自主命题
   - 2022：新高考改革继续
   - 2023：浙江停止自主命题
   - 2024：新高考改革继续
   - 2025：新高考改革完成

## 名词解释
***（为统一标准，命名方式可能与网络主流称呼，《试题分析》不同）***
- **全国卷**：存在两套课程标准时，年份较早的对应的*统一*高考试题，只存在一套时，指*统一*高考试题
- **新课程卷/大纲卷**：依照1996版课标（大纲）命制的统一高考试题  
  - 2000-2004 相对旧课程叫新课程卷  
  - 2005-2014 相对新课标叫大纲卷
- **旧课程卷**：存在新课程卷时，其余的统一高考试题，依照1986版课标命制
- **新课标卷**：依照2003版课标命制的统一高考试题
- **新高考卷**：依照2017版课标命制的统一高考试题（1993除外）
*（各统考卷具体使用省份无官方说明，有一定争议，不单独列出，见文件名和6.3）*

# 4. 高考制度建立以来课程标准、大纲演变：
   1. 1952 中学数学教学大纲（草案，1954、1956修订）
   2. 1963 全日制中学数学教学大纲（草案）
   3. 1978 全日制十年制学校中学数学教学大纲（试行草案）
   4. 1982 全日制六年制重点中学数学教学大纲（征求意见稿）
   5. 1983 高中数学教学纲要（草案）
   6. 1986 全日制中学数学教学大纲（1990、1994修订）
   7. 1996 全日制普通高级中学数学教学大纲（供试验用，2000、2002修订）
   8. 2003 普通高中数学课程标准（实验）
   9. 2017 普通高中数学课程标准（2020、2025修订）

# 6. 内容说明

## 1. 试卷收录标准
- 普通本科院校可参与招生
- **普通高中**学生均可参加
- 非院校直接招生 
- 春季高考加一条：难度与**当年**普通高考相当

据此，部分年份、省份的大专招生专用试卷，大学单独、联合招生试卷，近年山东，广东，天津，福建等省所谓~~*春季高考*~~试卷不收录

## 3. 缺失试题
1958剩余地区，1977不确定是否还有地区独立命题

## 3. 争议内容
2005海南省一说使用全国3卷

## 4. 关于副题
本仓库不收录**副题**，**无论是否启用过**，值得注意的是，2003未启用副题，2008四川，甘肃45区、市的延考区卷不是副题，副题又叫**备用卷**、**付题**（“付”为“副”的二简字）
## 5.关于预考题
本仓库不收录预考试题，但1977除外，因为只有1977预考和高考均由省级及以下单位命题；关于1977的黑龙江初试，江苏初考是否算作预考，这里采用[此观点](https://doi.org/10.13397/j.cnki.fef.2023.05.010)
